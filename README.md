# Agent & Skills 管理平台

一个低代码智能体创建和管理系统，允许用户通过可视化界面创建、配置和管理AI智能体及其技能组件。

## 功能特性

- 🤖 **智能体管理** - 创建和管理AI智能体
- 🛠️ **技能仓库** - 模块化技能组件管理
- ✨ **智能生成** - AI驱动的提示词生成
- 🎨 **可视化编辑** - 直观的拖拽式界面
- 📚 **提示词库** - 内置提示词模板库
- 🔧 **工具集成** - 支持MCP/MDP协议
